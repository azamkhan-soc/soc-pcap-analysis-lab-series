# 🧪 Lab 03 – Traffic Analysis Exercise (You Dirty Rat)

---

## 📌 Task Description

Analyze the provided PCAP file and generate a full incident report.

The report must include:

### 1. Executive Summary
Explain what happened in simple terms (who, what, when).

### 2. Victim Details
Identify:
- Hostname
- IP Address
- MAC Address
- Domain/user information

### 3. Indicators of Compromise (IOCs)
Extract:
- IP addresses
- Domains
- URLs (if present)
- Malware-related artifacts (if any)

---

## 🎯 Goal
To simulate real SOC analyst investigation using Wireshark and network forensics techniques.

---

## 🧠 Skills Practiced
- DNS analysis
- Kerberos authentication tracing
- TLS inspection
- Threat intelligence extraction
