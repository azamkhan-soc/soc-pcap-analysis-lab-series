# 🧪 Lab Task – NEMOTODES Traffic Analysis

## 🎯 Objective

Analyze the provided PCAP and alert logs to identify malicious activity in the NEMOTODES.HEALTH network.

---

## 🧠 Tasks

1. Identify infected Windows workstation  
2. Extract:
   - Hostname  
   - IP address  
   - MAC address  
   - User account  
3. Identify malicious domains  
4. Identify C2 infrastructure  
5. Correlate IDS alerts with PCAP evidence  
6. Document findings in a structured incident report  

---

## 📂 Provided Data

- PCAP file (network traffic capture)
- IDS alert logs
- Internal domain: NEMOTODES.HEALTH
- Network range: 10.11.26.0/24

---

## 🧪 Skills Practiced

- DNS analysis  
- Kerberos authentication analysis  
- SMB traffic analysis  
- C2 detection  
- IOC extraction  
- Incident reporting
