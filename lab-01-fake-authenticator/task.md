# Lab 01 – Task Description

## 🧪 Scenario
A user searched for Google Authenticator and downloaded a fake software installer from a malicious website.

This resulted in a suspected malware infection inside the network.

---

## 🎯 Objective

Analyze the provided PCAP file and identify:

1. IP address of the infected Windows client  
2. MAC address of the infected client  
3. Hostname of the infected system  
4. User account name used on the system  
5. Likely malicious domain used for download  
6. Command and Control (C2) IP addresses  

---

## 📂 Data Provided

- PCAP file: network traffic capture
- Windows domain environment (BLUEMOONTUESDAY)
- Internal network range: 10.1.17.0/24

---

## 🧠 Skills Tested

- DNS analysis
- Kerberos authentication analysis
- NBNS hostname discovery
- Traffic pattern analysis
- C2 detection
